3 SIMPLE STEPS TO GET STARTED:
================================================================================

STEP 1: Install XAMPP
----------------------
Download from: https://www.apachefriends.org/
Install and start Apache + MySQL services

STEP 2: Setup Database
----------------------
1. Go to: http://localhost/phpmyadmin
2. Create database: zips_chau_elibrary
3. Import file: database.sql

STEP 3: Copy Files
------------------
Copy the 'zips-chau-elibrary' folder to:
- XAMPP: C:\xampp\htdocs\
- WAMP: C:\wamp64\www\

DONE! Now visit: http://localhost/zips-chau-elibrary/

================================================================================
TROUBLESHOOTING:
================================================================================

Problem: "Cannot connect to database"
Solution: Make sure MySQL is running and check config.php

Problem: Blank page
Solution: Enable PHP error reporting or check Apache error log

Problem: Upload not working
Solution: Check folder permissions on uploads/ folder

Problem: Page not found
Solution: Verify files are in correct htdocs/www directory
